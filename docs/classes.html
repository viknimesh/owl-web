Classes




This is an example of how html is rendered. With the following options you can change almost every
class the way you want

Options

refreshClass

Type: String
Default: owl-refresh

Class during refresh.

loadingClass

Type: starting
Default: owl-loading

Class during load.

loadedClass

Type: String
Default: owl-loaded

Class after load.

rtlClass

Type: String
Default: owl-rtl

Class for right to left mode.
